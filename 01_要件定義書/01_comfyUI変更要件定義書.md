
5. GUI要件 (PVAI Studio)
本システムは運用者が直感的に操作できる管理GUI「PVAI Studio」を提供する。初心者が迷わずに操作できるよう、設定項目を極限までシンプル化する。

5.1 デザイン要件
- カラースキーム:
  - 背景色: 白 (#FFFFFF)
  - 文字色: 濃いグレー (#333333)
  - アクセントカラー: インディゴ (#4F46E5)
- レイアウト:
  - プロンプト入力エリアを最上部に配置し、最も目立たせる。
  - その直後にモード選択ボタンを配置する。
  - プロンプト入力エリア付近に「AIで具体化」ボタンを配置する。
  - 不要なパラメータ（Seed, Steps, 商品忠実度, Director AI設定など）はすべて排除する。

5.2 UX/機能要件
- 操作フロー:
  1. **プロンプト入力**: 生成したい動画のイメージをテキストで入力。
  2. **モード選択**: 「インフルエンサー」「リップシンク」「ポーズ制御」からボタンで選択。
  3. **プロンプト具体化 (任意)**: 「AIで具体化」ボタンを押すと、選択モードに合わせてプロンプトを高品質に書き換える。
  4. **動画生成**: ボタン押下後、現在のプロンプト（修正済み含む）で動画生成を開始。
- 状態フィードバック:
  - 生成ボタン押下時のローディング表示。
  - 成功/エラー時のトースト通知。
- プレビュー: 生成された動画を即座に再生。
- 結合機能: 複数の動画URLを入力し、ffmpegによる結合処理を呼び出し。

5.3 アーキテクチャ
- 構成: index.html (Frontend) + gui_server.py (Lightweight Backend)
- 通信: REST API (/api/generate, /api/concat, /api/enhance_prompt)
- バックエンド (Modal):
  - **動画生成モデル**: LongCat-Video (13.6B parameters) を採用。
  - **プロンプト拡張**: ユーザーの明示的な要求により、vLLM を用いて詳細な指示に変換する。
  - **推論エンジン**: Modal上のGPU (A10G/A100) で LongCat-Video を実行。
